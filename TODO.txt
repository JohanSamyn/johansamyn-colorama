Implementations
---------------

*ANSI escape characters* exist for colors, bold underline, moving cursor
and clearing screen.

*termcolor* is a module which provides a 'colored' method, that wraps a given
string with named escape characters. Meh.

*win32* seems to work.

http://support.microsoft.com/kb/101875
